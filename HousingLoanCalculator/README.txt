Application is mainly for calculating our housing loan EMI, Priciple, Interest.
Forecasting Tool.

Pre Requities:
	Install the mentioned software below.
		1. Java
		2. Eclipse IDE
		3. Jetty/Tomcat Server
		4. Git

To access the application:

Step1:  Checkout the code from the below path

Step2:  Import the \HousingLoanCalculator project into Eclipse by Right Click -> Import-> Existing Maven Projects

Step3:  Right click on HousingLoanCalculator -> Run as Jetty.

Step4:  Go to browser Hit http://localhost:8080/modules/Loan.html
